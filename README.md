# Sales-Performance-Dashboard
🚀 Overview

This project presents an interactive Sales & Profit Dashboard built using Power BI.
It helps analyze business performance across multiple dimensions such as time, category, customer behavior, and regional insights.

⸻

🎯 Key Features
	•	📌 KPI Cards (Total Sales, Profit, Orders, Quantity, Profit per Order)
  
  •	📈 Sales vs Profit Trend Analysis (Year-wise)
	
  •	📊 Category & Sub-Category Performance Breakdown
	
  •	🌍 State-wise Sales Distribution (Map Visualization)
	
  •	💳 Payment Mode Analysis
	
  •	👤 Top Customers Identification
	
  •	🎛 Interactive Slicers (Date, Category, Payment Mode)
